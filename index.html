========================================================================================================= Types of VCS: ========================================================================================================= 1. Centralized version control system 2. Distributed version control system 1. Centralized Version control system - ( Clearcase, SVN ) * No parallel work can be performed * Data loss in case some files are deleted or content is deleted * Connect to machine - > Bandwidth * Licence cost - $100 2. Distributed Version control system : (GIT/Github , bitbucket) - Subscription - $10 * Parallel work can be performed * No data loss, bec we get mechanism to download(clone) the entire repository * Less bandwidth bec one time connection only when push/clone/pull - Rest of time we can performed development offline. * Open Source - Free trail software - Source (code ) - Software is open for changes - We can customize the software, what ever code written in sodtware can be modified.
